USER FLOW SUMMARY (Text Description)
================================================

CLIENT JOURNEY:
---------------
1. Entry Point
   - User visits LegalLink
   - Sign up or Login
   - Select "Client" role

2. Finding Legal Help (Two Paths)
   Path A: Post a Gig
   - Create gig post (title, description, budget, timeline)
   - Receive applications from lawyers
   - Review applications and chat with lawyers
   
   Path B: Search for Lawyers
   - Browse lawyer profiles with filters
   - View profiles, reviews, and ratings
   - Contact lawyer via chat

3. Hiring & Payment
   - Select and hire lawyer
   - Fund escrow account with agreed amount (SECURE PAYMENT)

4. Case Collaboration
   - Secure messaging with lawyer
   - Document sharing
   - Track milestones

5. Project Completion
   - Review completed work
   - If satisfied: Approve milestone → Release escrow funds → Rate & review
   - If issue: Open dispute → Admin reviews → Resolution (refund or pay lawyer)

6. End: Case closed


LAWYER JOURNEY:
---------------
1. Entry Point
   - User visits LegalLink
   - Sign up or Login
   - Select "Lawyer" role
   - Submit credentials for verification
   - Wait for admin approval

2. Finding Work
   - Browse available client gigs
   - Submit proposals and applications
   - Optional: Subscribe to Pro plan for better visibility
   - Optional: Answer public Q&A questions to build reputation

3. Getting Hired
   - Wait for client to hire
   - Once hired, case is created
   - Define project milestones

4. Case Work
   - Communicate with client
   - Share documents
   - Update progress
   - Complete milestones

5. Payment & Approval
   - Request client approval for completed milestone
   - If approved: Receive payment from escrow
   - If disputed: Handle dispute → Admin resolution
   - Repeat for multiple milestones

6. Financial Management
   - View earnings dashboard
   - Withdraw funds to bank account

7. Continue: Return to finding more work


KEY FEATURES ILLUSTRATED:
-------------------------
✓ Role-based authentication (Client vs Lawyer)
✓ Lawyer credential verification by admin
✓ Two-way gig discovery (post or search)
✓ Secure escrow payment system
✓ Milestone-based project management
✓ In-app messaging and document sharing
✓ Dispute resolution mechanism
✓ Rating and review system
✓ Subscription plans for lawyers
✓ Earnings dashboard and payout system
✓ Q&A forum for reputation building
